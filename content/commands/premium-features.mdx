### Auto-Reply Integration

<Accordion title="Reply System Configuration">
**Command-based replies:** Trigger when specific commands are used
**Word-based replies:** Trigger on message content matching

<CodeGroup>

```bash Command Replies
=embed create {content} --reply: ban
=embed create {content} --reply: warn
=embed create {content} --reply: timeout
```

```bash Word Replies
=embed create {content} --reply: hello,hi --restricted: no
=embed create {content} --reply: help --restricted: yes
```

</CodeGroup>

<Info>
Restricted replies only trigger for users with Manage Messages permission, while unrestricted replies work for all users.
</Info>
</Accordion>
