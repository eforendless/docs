# AI Commands & Integration

Velveta integrates advanced AI capabilities through viAI for text generation, image creation, translation, and automated moderation analysis.

<Info>
All AI features use the viAI model and consume tokens based on your premium tier. View usage with `/tokens`.
</Info>

## Text Generation & Chat

<Accordion title="viAI Chat (Slash)">
**Command:** `/viai <prompt>`  
**Integration:** Guild (0), Bot DM (1), Private Channel (2)  
**Cooldown:** 10 seconds  
**Premium:** Token-based usage  

Advanced AI chat with context-aware responses and premium features.

<CodeGroup>

```bash Usage
/viai What is the best way to moderate a Discord server?
/viai Explain quantum computing in simple terms
/viai Write a poem about coding
```

```bash Features
• Context-aware conversations
• Premium tier token limits
• Rich embed responses
• Usage tracking and analytics
• Multi-language support
```

</CodeGroup>

<Warning>
Token limits per request: Free (200), Gold (500), Platinum (1000), Diamond (2000)
</Warning>
</Accordion>

<Accordion title="viAI Chat (Prefix)">
**Command:** `=viai <prompt>`  
**Cooldown:** 10 seconds  
**Premium:** Token-based usage  
**Category:** AI  

<CodeGroup>

## Image Generation

**Command:** `/generate <prompt> [private]`  
**Cooldown:** 30 seconds  
**Premium:** Gold+ required  
**Integration:** Guild (0), Bot DM (1), Private Channel (2)  

Generate high-quality images using advanced AI models.

<CodeGroup>

```bash Usage

/generate Cute cartoon cat wearing a wizard hat private:true

```

```bash Token Costs
• Gold: 1000 tokens per image
• Platinum: 800 tokens per image  
• Diamond: 600 tokens per image
• Higher tiers = better quality & speed
```

</CodeGroup>

Images are generated at 1024x1024 resolution with advanced prompt interpretation and style consistency.

</Accordion>
