# Context Menu Commands

Right-click interactions for quick access to user and message analysis features.

<Info>
Context menu commands appear in the "Apps" section when right-clicking on users or messages.
</Info>

## User Context Menu

<Accordion title="Check User Info">
**Command:** Right-click user → Apps → Check User Info  
**Premium:** Gold+ required  
**Permissions:** None  
**Integration:** Guild only  
**Cooldown:** 5 seconds  

Advanced user analysis with verification status, premium info, and risk assessment.

<CodeGroup>

```bash Information Displayed
• Account creation and age
• Server join date and tenure
• Verification status and passport
• Premium tier and token usage
• Moderation history summary
• AI risk assessment score
```

```bash Analysis Features
• Cross-server reputation tracking
• Behavioral pattern detection
• Escalation risk evaluation
• Premium status verification
• Role and permission analysis
```

</CodeGroup>

<Warning>
This command requires Gold+ premium tier and provides detailed information that should be used responsibly by moderators.
</Warning>
</Accordion>

## Message Context Menu

<Accordion title="Analyze Message">
**Command:** Right-click message → Apps → Analyze Message  
**Premium:** Diamond required  
**Permissions:** Moderate Members  
**Integration:** Guild only  

AI-powered message content analysis for moderation decisions.

<CodeGroup>

```bash Analysis Includes
• Content sentiment analysis
• Policy violation detection
• Threat level assessment
• Context evaluation
• Recommended actions
```

```bash Use Cases
• Ambiguous violation reports
• Escalation decision support
• Pattern recognition across messages
• Cultural context understanding
• Automated moderation validation
```

</CodeGroup>
</Accordion>

## Usage Guidelines

<Accordion title="Best Practices">

<CodeGroup>

```bash Moderation Context
• Use for complex cases requiring analysis

• Document significant findings
```

