## Command Types

Velveta Bot supports multiple command types to suit different user preferences:

- **Slash Commands** (`/command`) - Modern Discord commands with auto-complete
Velveta supports two command styles:
- **Context Menu** (Right-click) - Quick actions from user/message menus
- Slash commands: `/viai`, `/generate`, `/tokens`, `/premium`
- Prefix commands: `=warn`, `=timeout`, `=kick`, `=ban`, `=verification`
Essential commands for everyday use.

## Verification Commands

Manage server verification and security systems.

<AccordionGroup>
  <Accordion title="/verification setup" icon="shield-check">
    Configure verification system for your server.
    
    **Usage:**
    ```
    /verification setup channel:#verification method:phone
    /verification setup channel:#verify method:discord roles:@Verified
    ```
    
    **Methods available:**
    - `phone` - Phone number verification
    - `discord` - Discord account verification  
    - `captcha` - CAPTCHA challenges
    - `hybrid` - Multiple methods combined
    
    **Examples:**
    - `/verification setup channel:#verify method:phone`
    - `v!verify setup #verification discord`
  </Accordion>

  <Accordion title="/verification status" icon="clipboard-check">
    Check verification status of users.
    
    **Usage:**
    ```
    /verification status user:@username
    /verification status
    ```
    
    **Information displayed:**
    - Verification status
    - Method used
    - Verification date
    - Passport details
    - Risk assessment
  </Accordion>

  <Accordion title="/verification ban" icon="ban">
    Globally ban users from verification system.
    
    **Usage:**
    ```
    /verification ban user:@username reason:"Abuse of system"
    /verification unban user:123456789 reason:"Appeal approved"
    ```
    
    **Global ban features:**
    - Cross-server enforcement
    - Appeal system
    - Evidence tracking
    - Automatic detection
  </Accordion>

  <Accordion title="/verification reset" icon="refresh">
    Reset user verification status.
    
    **Usage:**
    ```
    /verification reset user:@username reason:"Technical issue"
    ```
    
    **Use cases:**
    - Technical problems
    - Method changes
    - False positives
    - User requests
  </Accordion>
</AccordionGroup>

## Utility Commands

Helpful tools and utilities for server management.

<AccordionGroup>
  <Accordion title="/poll" icon="chart-bar">
    Create interactive polls with reactions or buttons.
    
    **Usage:**
    ```
    /poll question:"What's your favorite color?" options:"Red,Blue,Green,Yellow"
    /poll question:"Movie night?" options:"Friday,Saturday" duration:1d
    ```
    
    **Features:**
    - Multiple choice options
    - Timed polls
    - Anonymous voting
    - Result analytics
    
    **Examples:**
    - `/poll question:"Server event time?" options:"Weekend,Weekday"`
    - `v!poll "Pizza or burgers?" "üçï,üçî"`
  </Accordion>

  <Accordion title="/remind" icon="clock">
    Set reminders for yourself or others.
    
    **Usage:**
    ```
    /remind time:1h message:"Check server logs"
    /remind time:tomorrow message:"Weekly meeting" user:@username
    ```
    
    **Time formats:**
    - Relative: `1h`, `30m`, `2d`
    - Absolute: `tomorrow`, `next friday`
    - Specific: `2024-12-25 10:00`
  </Accordion>

  <Accordion title="/role" icon="users">
    Manage user roles and permissions.
    
    **Usage:**
    ```
    /role add user:@username role:@RoleName
    /role remove user:@username role:@RoleName
    /role list user:@username
    ```
    
    **Bulk operations:**
    - Add roles to multiple users
    - Remove roles from inactive users
    - Role synchronization
    - Permission auditing
  </Accordion>

  <Accordion title="/channel" icon="hash">
    Channel management and utilities.
    
    **Usage:**
    ```
    /channel create name:"new-channel" type:text category:"General"
    /channel clone channel:#existing-channel name:"backup-channel"
    /channel slowmode channel:#general duration:5s
    ```
    
    **Channel operations:**
    - Create/delete channels
    - Clone channel settings
    - Manage permissions
    - Set slowmode
  </Accordion>
</AccordionGroup>

## Context Menu Commands

Right-click commands for quick actions.

### User Context Menu

Right-click any user to access these commands:

- **Warn User** - Quickly warn with reason prompt
- **Timeout User** - Apply timeout with duration selection  
- **Check Cases** - View user's moderation history
- **Analyze Risk** - Get user risk assessment
- **Copy User ID** - Copy user ID to clipboard

### Message Context Menu

Right-click any message for these options:

- **Report Message** - Report inappropriate content
- **Quote Message** - Quote in current channel
- **Archive Message** - Save to server archive
- **Translate Message** - Auto-translate text
- **Create Embed** - Convert message to embed

## Command Permissions

### Default Permissions

- **Everyone**: `/help`, `/ping`, `/info`, `/poll` (if enabled)
- **Manage Messages**: `/timeout`, `/warn`, `/cases`
- **Kick Members**: `/kick`, `/jail`
- **Ban Members**: `/ban`, `/unban`
- **Manage Server**: `/verification setup`, `/embed template`
- **Administrator**: All commands

### Custom Permissions

You can customize command permissions using Discord's built-in slash command permissions:

1. Go to **Server Settings** ‚Üí **Integrations** ‚Üí **Velveta Bot**
2. Find the command you want to modify
3. Add or remove roles/users
4. Set channel restrictions

### Permission Overrides

Some commands respect Discord's permission hierarchy:
- Moderation commands check role positions
- Users cannot moderate those with higher roles
- Bot role must be above target roles for role management
